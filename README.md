# 📖 Présentation

**PREDYNEX** est une plateforme intelligente de gestion de projets permettant aux chefs de projet et aux décideurs de suivre l'avancement des projets, anticiper les risques et prendre des décisions grâce à des modèles de Machine Learning.

L'application combine un frontend React, une API Node.js, une base de données MongoDB et un moteur IA développé en Python afin de fournir des analyses prédictives et des recommandations contextuelles.

# 📦 Installation

## Prérequis

- Node.js
- Python 3
- MongoDB Atlas
- Git

---

## 1. Cloner le projet

```bash
git clone https://github.com/semiawledmassoud/ai-project-management
.git

cd predynex
```

---

## 2. Installer le Backend

```bash
cd backend

npm install
```

Créer un fichier `.env`

```env
PORT=5000

MONGO_URI=Votre_URI_MongoDB

JWT_SECRET=Votre_Secret
```

---

## 3. Installer le Frontend

```bash
cd ../frontend

npm install
```

---

## 4. Installer le moteur IA

```bash
cd ../ia-engine

pip install flask flask-cors scikit-learn pandas numpy
```

---

# 🚀 Lancement

## Terminal 1

```bash
cd ia-engine

python app.py
```

---

## Terminal 2

```bash
cd backend

node server.js
```

---

## Terminal 3

```bash
cd frontend

npm start
```

L'application sera disponible sur :

```text
http://localhost:3000
```
